Test5 Component

The main purpose of this component to get the employees array from service class.
We are going to use service class because it is recommended that the component ts file only be used to support the view.
And, second thing if we create the array in our component itself, and it may be used to some other component, so how we are going to do that.
So, our two priciples of programming, if we are creating the array in component ts file.
1. DRY (Do not repeat yourself), if we are going to create same employees array in different component.
2. Single Responsibility Principle, as the component class is also providing data instead of only supporting view.
